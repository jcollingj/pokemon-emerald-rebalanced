# Eevee

**Build Identity:** A versatile support Pokemon that boosts its team with Agility/Bulk Up and passes the boosts via Baton Pass, or uses Adaptability Swift (120 effective BP, never misses) as a standalone attacker.

## Ability
**Adaptability** — STAB multiplier 2.0x instead of 1.5x. Swift = 120 effective BP. But the real fantasy is boosting and Baton Passing.

Set up Agility + Bulk Up, then Baton Pass to an Eeveelution sweeper. Or use standalone with Adaptability Swift for reliable never-miss damage.

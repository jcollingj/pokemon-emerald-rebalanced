# Ninjask

**Build Identity:** Ninjask is the fastest Pokemon in the game (160 Speed) that gets faster every turn via Speed Boost. It sweeps with escalating Bug/Flying STAB or passes compounding Speed + Attack boosts to teammates via Baton Pass.

## Ideal Items
1. **Focus Sash** — survive one hit, guarantee Swords Dance or Baton Pass setup
2. **Life Orb** — max damage output for outright sweeping
3. **Choice Band** — pure first-turn offensive pressure

## Core Strategy
Speed Boost makes Ninjask faster each turn; Swords Dance stacks attack. Either sweep with X-Scissor/Aerial Ace/Acrobatics, or Baton Pass accumulated boosts (+Speed +Attack) to a heavy teammate. Fury Cutter compounds in power each turn, synergizing with Speed Boost's escalating pressure.

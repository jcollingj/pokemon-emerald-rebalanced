# Skitty

## Build Identity
Normalize Special Sweeper pre-evolution. Introduces Normalize mechanic via Fake Out at L8 — priority + flinch that always hits as Normal STAB.

## Ability
Normalize (all moves → Normal type + 1.2× boost + Normal STAB = 1.8× effective on everything)

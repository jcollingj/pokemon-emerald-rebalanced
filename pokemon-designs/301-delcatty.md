# Delcatty

## Build Identity
Normalize Special Sweeper — every move in the kit hits at 1.8× effective power via Normalize (1.2× boost) + Normal STAB. Calm Mind turns it into a wincon; Return is the max-friendship payoff nuke.

## Ability
Normalize (all moves → Normal type + 1.2× boost + Normal STAB = 1.8× effective on everything)

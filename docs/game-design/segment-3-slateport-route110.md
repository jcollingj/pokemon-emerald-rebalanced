# Segment 3: Dewford → Slateport City → Route 110

**Route:** Dewford → Boat → Route 107/108/109 (water) → Route 109 Beach → Slateport City → Route 110 (Cycling Road + Trick House)

**Theme:** Ocean voyage into a bustling port city, Team Aqua confrontation at the museum, then rival rematch on the road north. The world opens up — new types (Electric, Poison), new story stakes, two boss-tier fights.

**Level cap:** L24 (Badge 2 from Brawly). No gym in this segment.

## Design Notes

### Difficulty
- Level cap L24 — player has evolved starters, Segment 1+2 team
- Both Aqua Grunts AND Rival are boss-tier fights
- This segment has no gym, so the boss fights ARE the challenge
- Route 110 trainers prep the player for Wattson (Electric types)

### Player's Team at This Point (~L20-24)
- Fully evolved starter (Sceptile/Blaziken/Swampert)
- 1-2 birds (Pidgeot, Noctowl, Swellow, Crobat)
- Fighting type from Dewford (Hariyama, Medicham, Heracross)
- Water type (Gyarados, Tentacruel, Azumarill)
- Possibly: Alakazam, Gardevoir, Sableye, Mawile, Aron

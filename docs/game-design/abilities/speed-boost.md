# Speed Boost

## Effect
At the end of each turn, the Pokemon's Speed stat increases by +1 stage. After one turn, Speed is 1.5x; after two turns, 2x. Snowballs into an uncatchable sweeper.

## Key Interactions
- Protect on turn 1 = free Speed boost with no risk
- After 1 boost: Blaziken outspeeds most of the game
- Pairs with Baton Pass to transfer Speed boosts to teammates

## Pokemon Using This
- **Torchic / Combusken / Blaziken** -- Physical kicking sweeper. Speed Boost fixes Blaziken's moderate base Speed, turning it into a snowball threat. Specializes in Fighting-type kicks (Blaze Kick, High Jump Kick). Can Baton Pass Speed boosts to teammates.

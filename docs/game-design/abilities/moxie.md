# Moxie

## Effect
When the Pokemon KOs an opponent, it gains +1 Attack stage. Creates a snowball effect where each kill makes the next hit harder.

## Key Interactions
- Dragon Dance (+1 Atk, +1 Spe) + first Moxie KO = +2 Atk / +1 Spe
- After two Moxie KOs: +3 Atk / +1 Spe -- Waterfall at 120 effective BP off 312 Atk
- Encourages sequencing: kill the easiest target first to fuel the snowball

## Pokemon Using This
- **Gyarados** -- Snowball setup sweeper. 125 Atk, 86 Speed (129 at +1). Dragon Dance provides the Speed, Moxie provides escalating Attack. Waterfall for STAB, Crunch/Ice Fang/Earthquake for coverage. Unlike Fearow (kamikaze) or Pidgeot (sustained), Gyarados invests a setup turn but rewards it with a sweep that gets stronger per KO.

# Sheer Force

## Effect
Moves with secondary effects (flinch, stat drops, status, etc.) lose those effects but deal 1.3x damage. Critical interaction: Life Orb recoil is suppressed on Sheer Force-boosted moves, giving a free 1.69x multiplier (1.3x x 1.3x).

## Key Interactions
- Life Orb + Sheer Force = 1.69x damage with ZERO recoil on boosted moves
- Forces meaningful move choices: Liquidation (Def drop, boosted) vs Crabhammer (high crit, not boosted)
- Crunch (Def drop, boosted to 104) vs Night Slash (high crit, not boosted at 70)

## Boosted Moves (Crawdaunt)
- Liquidation (85 -> 110, STAB -> 166)
- Crunch (80 -> 104, STAB -> 156)
- Rock Slide (75 -> 97, coverage)
- Bite (60 -> 78, STAB -> 117)
- Bubble Beam (65 -> 84, STAB -> 127)

## Pokemon Using This
- **Crawdaunt** -- Setup bruiser. Water/Dark with 120 Atk. Swords Dance + Life Orb + Sheer Force = recoil-free 1.69x damage. Aqua Jet patches 55 Speed with +1 priority. Unlike Sharpedo (fast glass cannon), Crawdaunt is slow, tanky enough to set up, and sweeps 2-3 Pokemon without chipping itself.

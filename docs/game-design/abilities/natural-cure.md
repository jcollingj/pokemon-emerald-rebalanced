# Natural Cure

## Effect
All status conditions (burn, poison, paralysis, sleep, freeze) are cured when the Pokemon switches out. Rewards pivot-heavy playstyles.

## Pokemon Using This
- **Roselia** -- Secondary ability (Ability 2). Backup option alongside Triage. Absorbs status and pivots out to cure, then returns clean.
- **Corsola** -- Secondary ability (Ability 2). Also rewards switching, pairs naturally with Corsola's Regenerator-style pivot play even without Regenerator.

# Intimidate

## Effect
On switch-in, lowers all opponents' Attack stat by -1 stage. Provides immediate defensive value without requiring any action.

## Pokemon Using This
- **Gyarados** -- Secondary ability (Ability 2). Defensive fallback for players who want Gyarados as a physical check rather than a Moxie sweeper. Doesn't shape the moveset like Moxie does, but provides reliable team support.

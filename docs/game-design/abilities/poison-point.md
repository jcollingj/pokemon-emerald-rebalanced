# Poison Point

## Effect
When the Pokemon is hit by a contact move, there is a 30% chance the attacker becomes poisoned.

## Pokemon Using This
- **Nidorina** -- Primary ability (Ability 1/2). Preview of Nidoqueen's poison synergy.
- **Nidorino** -- Primary ability (Ability 1). Contact poison fits the physical attacker pre-evolution.
- **Nidoqueen** -- Secondary ability (Ability 2). Alternative to Poison Heal for a contact-poison playstyle.
- **Roselia** -- Hidden ability. Thematic but not exciting compared to Triage.

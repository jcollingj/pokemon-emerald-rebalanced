# Vital Spirit

## Effect
The Pokemon cannot be put to sleep. Functionally identical to Insomnia.

## Pokemon Using This
- **Vigoroth** -- Primary ability. The hyperactive middle evolution of the Slakoth line. Unlike Slakoth (Truant) and Slaking (Truant reworked), Vigoroth acts every turn with no skip. Vital Spirit fits the "can't stop moving" personality.

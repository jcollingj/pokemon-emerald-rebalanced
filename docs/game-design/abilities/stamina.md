# Stamina

## Effect
Each time the Pokemon is hit by a damaging move, its Defense stat is raised by +1 stage. Creates a lose-lose for opponents: attacking makes the target tankier, not attacking lets it set up.

## Key Interactions
- Synergizes with Rollout builds: each hit while rolling makes Geodude harder to KO
- Multi-hit moves trigger Stamina multiple times
- Defense stacking makes Defense Curl + Rollout safer

## Pokemon Using This
- **Geodude / Graveler / Golem** -- Rollout tank. Stamina + Defense Curl + Rollout = snowball that gets tankier with every hit. Replaces Rock Head (which didn't shape the moveset since Rollout has no recoil).

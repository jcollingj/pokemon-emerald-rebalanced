# Technician

## Effect
Moves with base power of 60 or less receive a 1.5x damage boost. Turns weak utility moves into legitimate offensive options.

## Boosted Moves (Breloom)
- Mach Punch (40 -> 60, +1 priority Fighting STAB)
- Bullet Seed (25 x 2-5 -> 37.5 each, Grass STAB multi-hit)
- Force Palm (60 -> 90, 30% paralysis)
- Rock Tomb (60 -> 90, -1 Speed to target)

## Pokemon Using This
- **Breloom** -- Priority fighter. 135 Atk with Technician makes Mach Punch a devastating +1 priority move (60 effective BP off 135 Atk). Bullet Seed with Loaded Dice = 187.5 effective Grass STAB. Spore at L48 provides 100% sleep. The "weak moves hit like trucks" archetype.

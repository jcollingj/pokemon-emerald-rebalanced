# Sap Sipper

## Effect
Grants full immunity to Grass-type moves. When hit by a Grass move, the Pokemon's Attack stat is raised by +1 stage instead of taking damage.

## Pokemon Using This
- **Azumarill** -- Hidden ability. Covers the Grass weakness that threatens Water/Fairy typing. Each Grass attack absorbed gives +1 Atk, stacking with Huge Power's doubled Attack.

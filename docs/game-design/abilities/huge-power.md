# Huge Power

## Effect
Doubles the Pokemon's Attack stat. 50 base Atk becomes 100 effective; 55 base becomes 110 effective.

## Why It Works
Huge Power lets Azumarill hit like a 110 Atk Pokemon off a modest base stat, creating a wallbreaker that also has room for bulk (110 HP, 85/85 defenses). The Belly Drum combo pushes effective Attack to ~660, enabling Aqua Jet priority sweeps.

## Pokemon Using This
- **Azumarill** -- Physical Water wallbreaker. Water/Fairy typing gives excellent offensive coverage. Choice Band set hits 165 effective Atk. Belly Drum + Aqua Jet is the signature combo -- max Attack with priority sweep.

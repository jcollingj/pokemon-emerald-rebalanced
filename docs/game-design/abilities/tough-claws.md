# Tough Claws

## Effect
All contact moves deal 1.3x damage. Shapes the moveset: you pick contact moves over non-contact alternatives (X-Scissor over non-contact Bug moves, Stone Edge over Rock Slide).

## Boosted Moves (Armaldo)
- Stone Edge (100 -> 130, STAB -> 195)
- X-Scissor (80 -> 104, STAB -> 156)
- Aqua Tail (90 -> 117)
- Liquidation (85 -> 110)
- Brick Break (75 -> 97)
- Aqua Jet (40 -> 52, +1 priority)
- Metal Claw (50 -> 65)

## Pokemon Using This
- **Armaldo** -- Slow armored wallbreaker. 125 Atk with 1.3x on contact moves = massive physical damage. 85/100/80 bulk means it takes a hit before hitting back. Aqua Jet patches 45 Speed for priority cleanup. Choice Band Stone Edge at 195 effective BP off 187 Atk OHKOs most things.

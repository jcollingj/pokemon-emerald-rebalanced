# Sturdy

## Effect
If the Pokemon is at full HP, it survives any single attack with at least 1 HP remaining. A guaranteed Focus Sash effect built into the ability.

## Pokemon Using This
- **Geodude / Graveler / Golem** -- Secondary ability (Ability 2). Safety net for getting at least one Defense Curl off before Rollout starts.
- **Probopass** -- Secondary ability (Ability 2). Guarantees Stealth Rock gets set up even against super-effective hits.

# Adaptability

## Effect
Same-type attack bonus (STAB) is increased from 1.5x to 2x. Rewards using STAB moves over coverage.

## Pokemon Using This
- **Crawdaunt** -- Hidden ability. Alternative to Sheer Force that favors pure Water/Dark STAB over coverage moves. Crabhammer at 100 BP x 2.0 STAB = 200 effective, Night Slash at 70 BP x 2.0 STAB = 140 effective with high crit chance. The "crits build" variant.

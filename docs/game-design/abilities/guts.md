# Guts

## Effect
When the Pokemon has a status condition (burn, poison, paralysis, sleep, freeze), its Attack stat is boosted by 1.5x. Additionally, the burn damage reduction to Attack is negated -- Guts-boosted Pokemon hit at full power even when burned.

## Key Interactions
- Flame Orb = permanent 1.5x Atk boost (burn damage is 1/8 HP per turn)
- Toxic Orb = 1.5x Atk boost (escalating poison damage)
- Negates burn's Attack halving -- Flame Orb is pure upside for physical attackers
- Facade doubles to 140 BP when statused, stacking with Guts

## Pokemon Using This
- **Poochyena / Mightyena** -- Bulky physical attacker. Normal/Dark typing with Guts + Flame Orb for powered-up attacks. High HP (85) absorbs the burn chip.
- **Taillow / Swellow** -- Defensive tank (70/70/70 bulk). Guts + Flame Orb + Roost sustain. Unexpected tank that gets stronger when statused.
- **Raticate** -- Secondary ability option (Ability 2). Backup Flame Orb build alongside Strong Jaw.
- **Zigzagoon / Linoone** -- Hidden ability option. Alternative to the Pickup Belly Drum build.

# Merciless

## Effect
Attacks always land as critical hits against poisoned targets. The user must first poison the target (via Poison Powder, Poison Sting, etc.) to activate the guaranteed crits.

## Key Interactions
- Pin Missile with Merciless = potentially 5 critical hits in one turn
- Poison Powder sets up the combo for free
- Crits ignore the target's defense boosts

## Pokemon Using This
- **Beedrill** -- Crit sweeper. Poison Powder to poison, then Poison Jab/Pin Missile for guaranteed crits. Pin Missile (2-5 hits x 25 BP x crit) is potentially devastating. Swords Dance before attacking for massive damage. The condition-to-payoff pattern.

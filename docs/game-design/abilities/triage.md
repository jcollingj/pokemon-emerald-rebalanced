# Triage

## Effect
Moves that restore HP to the user gain +3 priority. This includes Giga Drain, Mega Drain, Absorb, Draining Kiss, Leech Seed, and Synthesis.

## Key Interactions
- Bypasses speed entirely on drain moves -- 65 Speed doesn't matter
- Priority Giga Drain (75 BP STAB) = attack first AND heal
- Priority Leech Seed = guaranteed to land before opponent moves
- Priority Synthesis = emergency 50% heal before getting hit

## Pokemon Using This
- **Roselia** -- Priority drain mage. 110 SpA with Triage makes Giga Drain a +3 priority 112.5 effective STAB that also heals. Leech Seed + Giga Drain + Triage = constant priority sustain loop. Big Root boosts drain healing to 65% of damage dealt. The "I heal faster than you hurt me" archetype.

# Strong Jaw

## Effect
Biting moves deal 1.5x damage.

## Boosted Moves
Bite, Crunch, Fire Fang, Ice Fang, Thunder Fang, Psychic Fangs, Hyper Fang, Poison Fang, Fishious Rend, Jaw Lock

## Pokemon Using This
- **Raticate** -- Fang coverage attacker. Every move in the kit is a biting move (Fire Fang, Ice Fang, Thunder Fang, Psychic Fangs, Crunch, Hyper Fang). Trades raw power for never being walled.
- **Sharpedo** -- Glass cannon STAB biter. Unlike Raticate's type coverage breadth, Sharpedo focuses on raw STAB bite power with Fishious Rend (Water STAB, doubles if faster = 382 effective BP) and Crunch (Dark STAB). Same ability, completely different identity.

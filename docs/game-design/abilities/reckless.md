# Reckless

## Effect
Moves that cause recoil damage deal 1.2x damage. The user takes more recoil in exchange for amplified offensive output.

## Boosted Moves
Brave Bird, Double-Edge, Take Down, Head Smash

## Pokemon Using This
- **Fearow** -- Kamikaze glass cannon (110 Atk, 105 Speed, 65/65/61 bulk). Both Brave Bird and Double-Edge get STAB + Reckless = 216 effective BP each. Head Smash reaches 180 effective. Fearow exists to delete 1-2 threats and go down swinging. The "I don't care if I die" bird.

# Keen Eye

## Effect
The Pokemon's accuracy cannot be lowered by opponent moves or abilities. Prevents Sand Attack, Flash, Smokescreen, and similar moves from reducing accuracy.

## Pokemon Using This
- **Pidgeot** -- Secondary ability (Ability 2). Flavor pick alongside Aerilate.
- **Fearow** -- Secondary ability (Ability 2). Backup option alongside Reckless.
- **Pelipper** -- Secondary ability (Ability 2). Backup option alongside Drizzle.

# Swift Swim

## Effect
Speed stat is doubled while rain is active. Transforms slow Pokemon into fast sweepers when paired with rain setters (Pelipper's Drizzle or Rain Dance).

## Key Interactions
- Omastar: 60 Speed -> 120 in rain, 240 after Shell Smash
- Poliwag: Rain Dance + Swift Swim = self-sufficient speed doubling
- Requires rain to function -- without it, these Pokemon are slow

## Pokemon Using This
- **Poliwag / Poliwhirl / Poliwrath** -- Rain sweeper. Swift Swim + Rain Dance for self-sufficient speed doubling. Special attacker focus with Water Pulse STAB.
- **Omastar** -- Rain Shell Smasher. Swift Swim fixes 60 base Speed, then Shell Smash pushes to 240 effective in rain. 115 SpA with rain-boosted Water moves = devastating.
- **Surskit / Masquerain** -- Rain pivot. Swift Swim + Tail Glow for special sweeping in rain. 90 SpA becomes 130 in rain at doubled speed.

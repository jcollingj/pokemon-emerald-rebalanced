# Battle Armor

## Effect
The Pokemon cannot be hit by critical hits. Functionally identical to Shell Armor.

## Pokemon Using This
- **Armaldo** -- Secondary ability (Ability 2). Roxanne's Anorith uses this with Focus Sash for the gym fight -- a defensive setup designed to survive and punish. The player's version uses Tough Claws for offense instead.

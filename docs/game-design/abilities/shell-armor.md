# Shell Armor

## Effect
The Pokemon cannot be hit by critical hits. Removes variance from defensive play -- incoming damage is always predictable.

## Pokemon Using This
- **Omastar** -- Secondary ability (Ability 2). Defensive backup for a bulkier set without relying on rain/Swift Swim.
- **Crawdaunt** -- Secondary ability (Ability 2). For players who want to tank hits rather than use Sheer Force offense.

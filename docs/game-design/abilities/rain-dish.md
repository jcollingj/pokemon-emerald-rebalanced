# Rain Dish

## Effect (Modified)
- Healing increased from 1/16 to **1/8 max HP per turn** in rain
- **+50% Defense** while rain is active (1.5x multiplier)

## Why Modified
Vanilla Rain Dish was too passive to define a build. The buffed version turns it into an active tank identity -- Mudkip/Swampert become rain fortresses with layered healing (Rain Dish + Aqua Ring) and boosted physical bulk.

## Pokemon Using This
- **Mudkip / Marshtomp / Swampert** -- Primary ability. Core rain tank identity. 85 HP Mudkip heals ~10 HP/turn in rain with 112 effective Defense.
- **Pelipper** -- Hidden ability. Synergizes with its own Drizzle for self-sustaining bulk.
- **Lotad / Lombre / Ludicolo** -- Primary ability. Rain support staller with drain moves and hazards.

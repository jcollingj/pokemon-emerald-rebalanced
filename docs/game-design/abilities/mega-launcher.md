# Mega Launcher

## Effect
Pulse and aura moves deal 1.5x damage. Boosts Water Pulse, Aura Sphere, Dragon Pulse, Dark Pulse, Terrain Pulse, Origin Pulse, and Hydro Cannon (flagged as pulse moves in this hack).

## Boosted Moves
- Water Pulse (60 -> 90, STAB -> 135)
- Aura Sphere (80 -> 120, never misses)
- Dragon Pulse (85 -> 127.5)
- Dark Pulse (80 -> 120, 20% flinch)
- Origin Pulse (110 -> 165, STAB -> 247.5)
- Hydro Cannon (150 -> 225, STAB -> 337.5)

## Pokemon Using This
- **Squirtle / Wartortle / Blastoise** -- Pulse mage coverage specialist. The "every move benefits" pattern applied to special attacks. Blastoise at 110 SpA with Mega Launcher has unmatched type coverage (Water/Fighting/Dragon/Dark) and late-game nuclear options in Origin Pulse and Hydro Cannon.

# Unburden

## Effect
When the Pokemon's held item is used up or removed, its Speed stat is doubled for the rest of the battle. One-time activation that transforms a moderate-speed Pokemon into an uncatchable sweeper.

## Key Interactions
- Sitrus Berry consumed at 50% HP -> Speed doubles -> sweep
- Power Herb consumed after a charging move -> Speed doubles
- Works with any consumable berry or item

## Pokemon Using This
- **Treecko / Grovyle / Sceptile** -- Physical Unburden sweeper. Consumes held item to double Speed, then sweeps with physical Grass, Dragon, and Fighting moves. The "I consume my item and go nuclear" archetype.

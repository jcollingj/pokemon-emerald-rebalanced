# Insomnia

## Effect
The Pokemon cannot be put to sleep. Blocks Hypnosis, Sleep Powder, Spore, and similar moves.

## Pokemon Using This
- **Noctowl** -- Secondary ability (Ability 2). Solid utility alongside Tinted Lens. Immune to the sleep moves that are common in this hack (Breloom's Spore, Noctowl's own Hypnosis).

# Segment 3 Trainers: Route 109 → Slateport → Route 110

## Design Rules
- All trainers have **2–4 Pokemon**
- Pokemon from **Segments 1–3 encounter pools**
- Competitive movesets — every trainer is a mini-puzzle
- Level range: L20–24 (player capped at L24)

- Brooke (Swimmer F) — Rain chain, weather counterplay
- Sam (Tuber M) — Belly Drum trap, priority awareness
